## Welcome to Viber

Viber is a speech-first coding experience where you describe what you want to build, an AI agent generates real code, applies it to a sandboxed workspace, and you immediately see the result in a live preview — all while the voice agent narrates what's happening.

Talk like you're speaking to another developer. No keyboard required for the primary workflow.

## How it works

Viber combines multiple AI services to create a seamless voice-to-code experience:

<Steps>
  <Step title="Speak your intent">
    Use natural language to describe the application you want to build or the changes you want to make.
  </Step>
  <Step title="AI understands and generates">
    Google Gemini analyzes your request, plans the architecture, and generates React code with Vite + TypeScript.
  </Step>
  <Step title="Code applies to sandbox">
    Generated files are automatically written to a Daytona sandbox workspace.
  </Step>
  <Step title="See it live">
    The preview updates instantly, and the voice agent narrates what's happening.
  </Step>
</Steps>

## Architecture overview

Viber's architecture combines voice interaction, code generation, and sandbox management:

- **User device**: Voice input + real-time UI preview
- **ElevenLabs voice agent**: Conversation loop, tool calls, narrated status updates
- **Google Gemini code agent**: Architecture planning and implementation
- **Daytona sandbox**: Runs the generated Vite app and serves a preview URL

<Note>
  Voice mode is optional. You can use Viber in text mode without configuring ElevenLabs.
</Note>

## Tech stack

Viber is built with modern web technologies:

- **Frontend**: React 19, TanStack Start, Vite, TypeScript, Tailwind CSS
- **Voice**: ElevenLabs Conversational AI
- **Code generation**: Google Gemini (gemini-3-pro)
- **Sandbox**: Daytona SDK
- **Images**: Unsplash API
- **Runtime**: Bun
